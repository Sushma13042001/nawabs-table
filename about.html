Data Architecture
Logical Data Model

The Nawab's Table data model has four entities with two one-to-many (1:N) relationships: one Customer can place many Orders, and one Order can contain many Order Items. Each Order Item links to one Menu Item (dish). Primary keys are underlined; foreign keys are labelled FK.

Relationship 1: CUSTOMERS (1) → ORDERS (N) — one customer can have many orders.
Relationship 2: ORDERS (1) → ORDER_ITEMS (N) — one order contains many dish items.
MENU_ITEMS is referenced by ORDER_ITEMS to retrieve dish details like price and name.
